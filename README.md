# Kleinanzeigen Plus

Eine Browser-Extension zur Verbesserung der Nutzererfahrung auf Kleinanzeigen.de.

## Funktionen

*   **🚫 User-Blocking**: Blockiere unerwünschte Verkäufer direkt in der Suche oder auf der Detailseite.
*   **👁️ Ad-Hiding**: Blende einzelne Anzeigen manuell aus, um die Suchergebnisse sauber zu halten.
*   **✅ Gesehen-Markierung**: Erkennt automatisch bereits besuchte Anzeigen und markiert diese in der Suchliste.
*   **📊 Statistik & Verwaltung**: Behalte den Überblick über blockierte User und ausgeblendete Anzeigen im Popup-Menü.

## Installation

1. Lade diesen Ordner herunter.
2. Öffne in Chrome/Edge `chrome://extensions`.
3. Aktiviere den "Entwicklermodus" (oben rechts).
4. Klicke auf "Entpackte Erweiterung laden" und wähle diesen Ordner aus.
