# Privacy Policy – Kleinanzeigen Plus

**Kleinanzeigen Plus** does not collect, transmit, or share any personal data.

## Data Storage

All data saved by this extension is stored **exclusively on the user's local device** using the browser's built-in `chrome.storage.local` API. This includes:

- Extension settings (filters, feature toggles)
- Lists of hidden ad IDs and blocked seller IDs
- Personal notes per listing
- Saved ad templates
- Chat reply snippets

## Network Access

The extension accesses `kleinanzeigen.de` solely to enhance the browsing experience:
- **Infinite Scroll**: Fetches additional result pages from kleinanzeigen.de to append more listings
- **Image Previews**: Loads listing detail pages from kleinanzeigen.de to display all photos on hover

No data is sent to any external server. No analytics, no tracking, no advertising.

## Downloads

The optional image download feature saves listing images directly to the user's local device. No data is uploaded anywhere.

## Third Parties

No user data is sold, shared, or transferred to any third party.
